## 게이트 미충족 시 대응

베타 출시 일정 압박이 있어도 게이트 미충족 상태에서 출시하지 않는다.

대안 옵션:
1. **시연 데모데이로 한정**: 사용자 모집 없이 데모 데이터로 데모데이 진행 → 사용자 실증 단계 연기.
2. **G6 미충족 시**: 룰 엔진 비활성화 + 「검산 가능 케이스 표시 후 룰 엔진 활성화 안내」로 부분 베타.
3. **G3 미충족 시**: `ENABLE_PDF_EXPORT=false`로 PDF 다운로드 비활성화 + 마스킹본 Markdown만 사용자에게 제공.
4. **G5 미충족 시**: 메시지 예시 페이지(`pages/7`) 비활성화 + 자료 확인 안내만 정적 안내.

정책 07 §1 — "개발 속도보다 출시 통제력이 경쟁력이다."
